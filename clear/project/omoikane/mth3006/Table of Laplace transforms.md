# Table of Laplace Transforms

| Function         | Laplace transform                   |
| ---------------- | ----------------------------------- |
| $e^{-at}$        | $\frac{1}{s+a}$                     |
| $\sin(\omega t)$ | $\frac{\omega}{s^2 + \omega^2}$     |
| $\cos(\omega t)$ | $\frac{s}{s^2 + \omega^2}$          |
| $t^n, n \geq 0$  | $\frac{n!}{s^{n+1}}$                |
| $t^p, p > 1$     | $\frac{\Gamma(p+1)}{s^{p+1}}$       |
| $y'$             | $s \tilde{y}(s) - y(0)$             |
| $y''$            | $s^2 \tilde{y}(s) - s y(0) - y'(0)$ |
